# UX Telemetry Simulation Project

## Objective
Simulate UX telemetry for a healthcare appointment app, measure drop-offs, engagement, and feature adoption.

## Methodology
1. Defined user journey & key events
2. Generated simulated user event data (100+ users)
3. Built Power BI dashboards for funnel analysis, feature engagement, device distribution, and time-on-task
4. Generated actionable insights for UX improvement

## Insights
- Signup completion: 56%, major drop-off between sign up start and sign up completion
- Feedback submission low → recommend prompts/reminders
- Mobile users have a higher percentage of appointment completion → web UI optimization
- Patients have a lower sign up completion than doctors → UI might not be optimized for patients. Further usability testing may be required.
